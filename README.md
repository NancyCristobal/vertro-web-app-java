### Descripcion
Sitio Web Informativo que ayudara a proveedores y clientes a ubicar el negocio fisicamente, asoi como conocer el numero de telefono y ver el catalogo de productos que ofrece el negocio.

### Problema Identificado
- Se identifico que el cliente no cuenta con una presencia digital y busca expandirse a la web.

### Solucion
- Crear un sitio web iformativo acorde a la necesidad del cliente.


### Documentacion
Vertro es una empresa local enfocada en el reciclaje de envases en la zona metropolitana de Monterrey, N.L. Nuestro objetivo es facilitar el reciclaje a negocios mediante un punto organizado.
